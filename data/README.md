# Dataset

**Source:** [Brain MRI Images for Brain Tumor Detection](https://www.kaggle.com/datasets/navoneel/brain-mri-images-for-brain-tumor-detection)

**Download instructions:**

1. Go to the Kaggle link above
2. Download and extract the ZIP
3. Place the extracted folders here:
```
data/
├── yes/    ← MRI images WITH tumor (155 images)
└── no/     ← MRI images WITHOUT tumor (98 images)
```

**Dataset statistics:**

| Class    | Count | After Augmentation |
|----------|-------|--------------------|
| Tumor    | 155   | 1,000              |
| No Tumor | 98    | 1,000              |
| **Total**| **253**| **2,000**         |
